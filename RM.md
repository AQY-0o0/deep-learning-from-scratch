## Python和外部库
需要以下软件来执行源代码。

* Python 3.x
* NumPy
* Matplotlib

* Python版本使用3系列。

##执行方法

转到每个章节的文件夹，然后运行Python命令。

`````
$ cd ch01
$ python man.py

$ cd ../ch05
$ python train_nueralnet.py
`````
